    LiveBroadcast

    Automatic Broadcast Plugin for CraftBukkit

    The "title" is the custom name that appears before each message. "timer" is the time in seconds between each message. You can have as many messages as you would like, the only limit is your servers processing power.

    If you would like to use color-codes use the '&' symbol followed by 0-9 and a-f. For more information on color-codes visit, http://www.minecraftwiki.net/wiki/Formatting_codes

    If you use a color-code ensure that the string is surrounded by single-quotes (' ')

    EXAMPLE:

    INCORRECT: 1: This &1is &fa test of &6the color-codes

    OUTPUT: This &1is &fa test of &6the color-codes

    CORRECT: 1: 'This &1is &fa test of &6the color-codes'

    OUTPUT: This is a test of the color-codes

    Permissions:

    lb.*:
  description: Gives access to all of the LiveBroadcast Commands
  children: 
    lb.broadcast: false
    lb.config.*: false
    lb.credits: false
    lb.reload: false
    lb.toggle: false
  default: false
  lb.broadcast:
    description: Allows the player to broadcast a custom message to the server
    default: false
  lb.config.*: 
    description: Allows the player to edit the config.yml
    children:
      lb.config.add: false
      lb.config.del: false
      lb.config.list: false
    default: false
    lb.config.add:
        description: Allows the player to add messages to the config.yml
        default: false
    lb.config.del:
        description: Allows the player to remove messages from the config.yml
        default: false
    lb.config.list:
        description: Allows the player to view the messages in the config.yml
        default: false
  lb.credits:
    description: Allows the player to view the plugin creator and version
    default: true
  lb.reload:
    description: Allows the player to reload the config.yml
    default: false
  lb.toggle: 
    description: Allows the player to turn LiveBroadcast on and off
    default: false
